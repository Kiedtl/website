linux @ kmach

My system is Manjaro v18.0, a user-friendly version of Arch Linux. It uses i3-gaps with a custom configuration and i3locks-fancy as its lock mechanism. For the terminal, I use Alacritty with ZSH and sometimes PowerShell. For the shell prompts, I use Oh-My-ZSH for ZSH and Pshazz for PowerShell.

I use polybar for the status bar and Vivaldi Snapshot as my browser.

Running three instances of Vivaldi and one instance of Dolphin and Alacritty each, my CPU usaage is surprisingly light - a maximum of 25% usage and a minimum of 15% usage. RAM usage is around 5500 MB and the core temperature is less than 40°C.

It sounds strange, but I use three different text editors, depending on what I am editing. For .NET code HTML, CSS, JavaScript, and other miscellaeneous tasks, I use Visual Studio Code. For Rust, Shell scripts, PowerShell, I use Spacemacs. For very quick tasks where I don’t like to waste time starting up the editor, I use SpaceVim.

I’ve also configured i3-gaps to automatically set the wallpaper and terminal colors on startup. I have a directory of wallpapers and use pywal to do the job.
